# m-site-guide

## 项目简介

`m-site-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不绑定任何特定域名或网站，仅作为静态页面文件的存放与展示用途。  
所有页面内容均以独立 `.html` 文件的形式组织，便于直接浏览或嵌入其他环境。

## 目录说明

```
m-site-guide/
├── index.html          # 仓库入口页面（导航或说明页）
├── pages/              # 存放主要 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库的默认首页，建议作为页面索引或简单说明。
- `pages/`：所有独立 HTML 页面均放置于此目录下，每个文件对应一个页面。
- `assets/`：存放页面引用的样式表、脚本、图片等公共资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面之间可通过相对路径相互链接，但建议保持各自独立性。
- 如需新增页面，直接在 `pages/` 目录下创建 `.html` 文件，并按需更新 `index.html` 的导航链接。
- 资源文件应统一存放于 `assets/` 目录，避免分散在页面文件夹中。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请遵守文件命名规范（建议使用英文小写及连字符）。
- 请勿在页面中包含任何违反平台政策的内容。
- 提交更改前建议本地预览，确保页面显示正常。
- 若需删除或移动页面，请同步更新相关引用链接，避免死链。

## 许可

本仓库内容仅供学习与存档参考，不构成任何形式的承诺或服务。  
具体页面可能引用外部资源，请各自遵守相应授权。
